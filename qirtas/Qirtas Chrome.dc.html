GTK / CSS NOTES
• Search: entry.search { border-radius: 999px; padding: 9px 16px; } and entry.search:focus { box-shadow: 0 0 0 3px alpha(@accent,.13); border-color: @accent; }
• Menu: build with GtkPopoverMenu; add class .qirtas-menu and style modelbutton { border-radius: 9px; padding: 8px 11px; }, popover.menu contents { border-radius: 14px; padding: 7px; }. Tint the Quit row via a .destructive class.
• Switches: replace the GtkCheckButtons with GtkSwitch; style switch { min-width: 40px; min-height: 23px; border-radius: 999px; } and switch:checked { background: @accent; }.
• Theme control: a GtkBox.linked of two GtkToggleButtons (Light/Dark) instead of the dropdown.
• Add @define-color accent per theme (#213A63 navy / #C9A86B dark) so all three reuse one token.
