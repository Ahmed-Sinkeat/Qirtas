# قرطاس

محرر ملاحظات خفيف مبني بلغة Zig.

قرطاس محاولة لجعل برنامج الملاحظات يعود برنامج ملاحظات فقط.

الهدف بسيط:

* كتابة الملاحظات.
* قراءة الملاحظات.
* تنظيم الملاحظات.
* تصدير الملاحظات.
* امتلاك الملاحظات.

قبل إضافة أي ميزة يُطرح سؤال واحد:

«هل تجعل هذه الميزة قرطاس برنامج ملاحظات أفضل؟»

إذا كانت الإجابة نعم، فقد تدخل إلى المشروع.

إذا كانت الإجابة لا، فإما أن تكون إضافة مستقلة أو لا تُضاف إطلاقاً.

## المبادئ

* ملفات Markdown عادية.
* دعم قوي للغة العربية.
* لا اشتراكات.
* لا حبس للبيانات.
* استخدم خدمة المزامنة التي تريدها.
* تطبيق أصلي وخفيف.
* لا ذكاء اصطناعي داخل التطبيق.

## الحالة الحالية

قرطاس ما زال في مرحلة ما قبل 1.0 ويتطور بصورة مستمرة.

## ما أعمل عليه حالياً

* تحسين تجربة الكتابة.
* تطوير دعم العربية.
* تحسين التصدير والقوالب.
* دعم أفضل للجداول.
* نسخة للهواتف.
* استيراد وتصدير DOCX.
* نظام إضافات.

## ما لا أنوي إضافته إلى النواة الأساسية

* الذكاء الاصطناعي.
* Graph View.
* أي ميزة لا تحسن تجربة الكتابة أو القراءة أو تنظيم الملاحظات بشكل مباشر.
